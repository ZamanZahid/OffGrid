# iOS app scaffold

This folder contains an Xcode-ready SwiftUI app scaffold for the same Relay + AeroGaze experience that exists in the Android app.

## Open in Xcode

1. Open the folder [ios-app](ios-app) in Xcode.
2. Open the project file [ios-app/OffgridIOSApp.xcodeproj](ios-app/OffgridIOSApp.xcodeproj).
3. Select the OffgridIOSApp target and run on an iPhone or simulator.

## Notes

- The app sources live in [ios](ios) and are referenced by the Xcode project.
- The relay networking code and the SwiftUI views already mirror the Android app flow.
- The Skygaze screen is a native iOS landing view for the offline star-solver experience.
